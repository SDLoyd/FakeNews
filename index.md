# Fake News

<img src="images/fakenews.jpg" width ="500">

## Description

The purpose of this project is to analyze a Kaggle data set and to create a predictive model to label fake news.

## Data

The Kaggle [data](https://www.kaggle.com/clmentbisaillon/fake-and-real-news-dataset) provided by [Cl√©ment Bisaillon](https://www.kaggle.com/clmentbisaillon).
